Description: The Pacmen Exercise project is a fun and interactive web application inspired by the classic Pac-Man game. Users can control amount of Pac-Men characters using the ui buttons. The application utilizes HTML, CSS, and JavaScript to implement game mechanics such as movement and collision detection. It offers a nostalgic gaming experience while showcasing fundamental programming concepts and web development skills.

Description: The Eye Exercise project is a captivating web experiment that showcases the power of CSS and JavaScript in creating interactive user experiences. In this project, a pair of eyes follow the user's mouse cursor as it moves across the screen, giving the illusion of being watched. The project demonstrates fundamental concepts of event handling, DOM manipulation, and CSS animation techniques. It serves as a playful demonstration of front-end web development skills while providing an engaging user interaction.

Description: The BusTracker project is a real-time web application designed to provide users with live tracking of bus routes and arrival times. Leveraging modern web technologies such as HTML, CSS, JavaScript, and APIs, BusTracker fetches data from transit agencies to display the current location and estimated arrival times of buses on selected routes. Users can easily search for routes, view bus positions on an interactive map, and plan their journeys with confidence. This project demonstrates the practical application of web development skills in creating useful and efficient transportation solutions.
